#创建订单
order/submit | post
> 参数
* orderId 预定单ID
* shippingAddressId 收货地址ID
* shippingFee 运费

> 返回  
* code 状态码
* msg 状态信息
* {seller:object,buyer:object,commodity:object,count:int,order_id:string}

#订单详情
order/detail | post
> 参数  
* id 订单号

> 返回  
* code 状态码
* msg 状态信息
* data Order viewModel

#订单列表
order/orderlist | get
> 参数  
* userId 用户id
* userType 1 买家 2 卖家
* status 1 待支付 2 待发货 3 待收货 4 待评论 5 已完成 6 退款中 7 已删除  不传则是查询全部
* timeline (可选)
* timeType 1-时间点后的新消息 2-时间点前的历史消息 (可选)


> 返回  
* code 状态码
* msg 状态信息
* data OrderList viewModel
