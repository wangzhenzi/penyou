#保存评论
comment/add-comment | post
> 参数  
* targetId 商品或文章ID
* userId 用户ID
* replyId 被回复的评论ID,为空则是独立的评论
* text 评论内容
* type 1-商品评论 2-文章评论
* pics 上传图片(可选)

> 返回  
* code 状态码
* msg 状态信息
* data 评论对象

#保存评价
comment/add-evaluation | post
> 参数
* userId 用户ID
* orderId 订单号
* replyId 被回复的评论ID
* text 评价内容
* sellerEvaluation 卖家评分
* commodityEvaluation  商品评分
* pics 上传图片(可选)

> 返回  
* code 状态码
* msg 状态信息
* data 评论对象


#删除评论
comment/delete-comment | post
> 参数  
* id 评论ID

> 返回  
* code 状态码
* msg 状态信息


#获取评论
comment/comments | get
> 参数  
* type 1-商品评论 2-文章评论
* targetId 商品或文章ID
* timeline 时间点(可选)
* timeType 1-返回大于时间点数据 2-返回小于时间点数据

> 返回  
* code 状态码
* msg 状态信息
* data 评论对象列表
