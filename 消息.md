#发送消息
message/send-message | post
> 参数  
* receiverId 接收者ID
* userId 发送者ID
* content 消息内容
* pics 消息图片地址 (可选)
* action 跳转操作{appType:ios/android,params:""} (可选，为空是聊天消息)

> 返回  
* code 状态码
* msg 状态信息

#消息列表
message/messages | get
> 参数  
* userId 用户ID
* timeline 时间点 (可选)

> 返回  
* code 状态码
* msg 状态信息
* data 消息对象列表

#发送和接收消息列表
message/send-recve | get
> 参数  
* userId 用户1 ID
* personId 用户2 ID
* timeline 时间点 (可选)

> 返回  
* code 状态码
* msg 状态信息
* data 消息对象列表
