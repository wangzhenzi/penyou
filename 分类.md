* cateId 分类ID

> 返回  
* code 状态码
* msg 状态信息

#取消关注分类
category/cancel-favorite | post
> 参数  
* userId 用户ID
* cateIds 多个分类id用逗号隔开

> 返回  
* code 状态码
* msg 状态信息

分类的商品
category/commodity | GET
> 参数  
* cateId 分类id
* timeline 时间点
* timeType 1(时间点之后)|2(时间点之前)
* orderby time|price|hot(暂不支持热度排序)
* asc 1 升序 0/null 降序

> 返回  
* code 状态码
* msg 状态信息
* data [PMiddleCommodity]
